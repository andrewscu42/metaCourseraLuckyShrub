# metaCourseraLuckyShrub

This project focuses on modernizing and optimizing the Lucky Shrub web application by implementing responsive design techniques. The goal was to enhance flexibility by adding media queries and ensuring the assets render correctly across devices.

## Features

- **Media Queries:** Adapt layouts and content for various screen sizes.
- **Flexible Assets:** Images and other media scale seamlessly.
- **Responsive Design:** User experience remains consistent on desktop, tablet, and mobile.

## Technologies Used

- HTML
- CSS (with responsive media queries)

## Getting Started

No installation required. Simply open `index.html` in any web browser to view the responsive site.
